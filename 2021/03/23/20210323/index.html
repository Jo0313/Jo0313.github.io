  • ggplot2와 dplyr 패키지 설치 방법을 기재하세요.

문제 3. 엑셀 데이터를 불러오세요.

문제 4. private_work, unemployment를 활용하여 산점도를 작성하세요.

문제 5. dplyr 함수를 활용하여, 아래 데이터를 요약하세요.

  • counties 데이터를 활용합니다.

  • 각 region, state 별 men, women 전체 인구수를 구합니다.
  • 최종 데이터셋 저장 이름은 final_df로 명명합니다.
